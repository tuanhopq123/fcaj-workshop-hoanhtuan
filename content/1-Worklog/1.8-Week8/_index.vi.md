## Mục tiêu tuần

* Hoàn thiện toàn bộ tài liệu Workshop.
* Bổ sung quy trình kiểm thử API và kết quả xác thực hệ thống.
* Cập nhật kiến trúc hệ thống với các giải pháp bảo mật bằng AWS KMS và AWS CloudTrail.
* Hoàn thành các bài thực hành về Amazon RDS, AWS KMS, CloudTrail, Athena và IAM Roles.

## Kết quả đạt được

* Hoàn thiện gần như toàn bộ tài liệu Workshop.
* Bổ sung đầy đủ quy trình kiểm thử API và hình ảnh minh họa.
* Hoàn thiện kiến trúc bảo mật với AWS KMS và CloudTrail.
* Hiểu rõ sự khác biệt giữa Amazon RDS và Amazon DynamoDB.
* Nâng cao kiến thức về IAM Roles, mã hóa dữ liệu và kiểm toán hệ thống trên AWS.

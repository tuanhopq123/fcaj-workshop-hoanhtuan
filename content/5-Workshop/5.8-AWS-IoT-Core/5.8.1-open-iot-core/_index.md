1. In the AWS Console search bar, type: `IoT Core`
2. Choose: **IoT Core**
3. In the left menu, find the group: **Manage**
4. Choose: **Message routing**
5. Then choose: **Rules**

{{% notice note %}}
According to the current AWS interface, the path to create a rule is **AWS IoT Core → Message routing → Rules**.
{{% /notice %}}

6. You will enter the Rules page. Click: **Create rule**

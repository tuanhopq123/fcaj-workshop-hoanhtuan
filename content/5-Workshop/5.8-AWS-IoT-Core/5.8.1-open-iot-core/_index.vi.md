1. Tại thanh tìm kiếm AWS Console, nhập: `IoT Core`
2. Chọn: **IoT Core**
3. Trong menu bên trái, tìm nhóm: **Manage**
4. Chọn: **Message routing**
5. Chọn tiếp: **Rules**

{{% notice note %}}
Theo giao diện AWS hiện tại, đường dẫn tạo rule là **AWS IoT Core → Message routing → Rules**.
{{% /notice %}}

6. Bạn sẽ vào trang Rules. Bấm: **Create rule**

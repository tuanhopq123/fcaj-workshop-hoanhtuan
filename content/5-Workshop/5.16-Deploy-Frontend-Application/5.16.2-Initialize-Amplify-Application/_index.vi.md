#### Bước 1: Truy cập AWS Amplify

1. Tại thanh tìm kiếm của AWS Console, nhập: `Amplify`
2. Chọn dịch vụ **AWS Amplify**.
3. Nhấn vào nút **Create new app** (hoặc **Start building with Amplify** nếu đây là lần đầu bạn sử dụng dịch vụ).

#### Bước 2: Chọn Nhà cung cấp Mã nguồn (Source Code Provider)

1. Tại trang **Start building with Amplify**, di chuyển xuống mục **Deploy your app**.
2. Chọn **GitHub** (hoặc nền tảng Git tương ứng nơi bạn đang lưu trữ mã nguồn của ứng dụng Frontend).
3. Nhấn **Next** để chuyển sang bước xác thực tài khoản và phân quyền truy cập kho lưu trữ (repository).

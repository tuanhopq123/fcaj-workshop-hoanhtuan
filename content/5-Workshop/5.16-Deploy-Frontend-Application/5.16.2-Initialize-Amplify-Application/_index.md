#### Step 1: Access AWS Amplify

1. In the AWS Console search bar, enter: `Amplify`
2. Select the **AWS Amplify** service.
3. Click the **Create new app** button (or **Start building with Amplify** if it is your first time).

#### Step 2: Choose Source Code Provider

1. On the **Start building with Amplify** page, navigate to the **Deploy your app** section.
2. Select **GitHub** as your Git provider where your frontend project code repository is hosted.
3. Click **Next** to proceed to repository authorization.

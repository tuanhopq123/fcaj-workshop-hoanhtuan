Before storing energy waste data, we need to create a DynamoDB table to hold room profiles and sensor data.

1. **Search for DynamoDB**
   * In the AWS Console search bar, type `DynamoDB`.
   * Select the **DynamoDB** service.

2. **Open Tables**
   * In the left menu, click **Tables**.

3. **Start creating a table**
   * Click **Create table**.

Cognito can automatically name the user pool based on the application[cite: 14]. If the name is incorrect, change it to: `energy-waste-user-pool`[cite: 14].

1. On the user pool page, choose: **Overview**[cite: 14].
2. Click: **Rename**[cite: 14].
3. Enter: `energy-waste-user-pool`[cite: 14].
4. Click: **Save changes**[cite: 14].
5. The Overview page should display **User pool name: energy-waste-user-pool** and **Region: ap-southeast-1**[cite: 14].

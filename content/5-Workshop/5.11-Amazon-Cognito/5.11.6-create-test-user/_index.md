1. In the user pool menu, choose: **Users**[cite: 14].
2. Click: **Create user**[cite: 14].
3. For **User name** and **Email address**, enter the email you use to receive test emails (e.g., `your-email@example.com`)[cite: 14].
4. Check the box for: **Mark email address as verified** (if available)[cite: 14].
5. For **Temporary password**, choose **Generate a password** or manually enter a temporary password (e.g., `Energy@2026Test`)[cite: 14].
6. In the invitation section, choose: **Send an email invitation**[cite: 14].
7. Click: **Create user**[cite: 14].

After creation, the user might display the status: **Force change password**[cite: 14]. This is the normal status before the first login[cite: 14].

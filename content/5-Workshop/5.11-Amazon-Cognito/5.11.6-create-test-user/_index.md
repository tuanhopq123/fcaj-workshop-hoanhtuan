1. In the user pool menu, choose: **Users**.
2. Click: **Create user**.
3. For **User name** and **Email address**, enter the email you use to receive test emails (e.g., `your-email@example.com`).
4. Check the box for: **Mark email address as verified** (if available).
5. For **Temporary password**, choose **Generate a password** or manually enter a temporary password (e.g., `Energy@2026Test`).
6. In the invitation section, choose: **Send an email invitation**.
7. Click: **Create user**.

After creation, the user might display the status: **Force change password**. This is the normal status before the first login.

---
title: "5.18.7. Delete Four Lambda Functions"
date: 2026-07-07
weight: 7
---

1. Search for **Lambda** in the AWS Console search bar and open the service.
2. Select **Functions** from the left navigation pane.
3. Check the boxes next to the following four project functions:
   * `energy-waste-detector`
   * `energy-virtual-sensor`
   * `energy-api-handler`
   * `energy-report-generator`
4. Click the **Actions** dropdown menu and select **Delete**.

![Select Lambda Functions](/images/5-Workshop/5.18-CleanUp/Picture1.png)

5. Type the confirmation string required by the AWS dialog box and click **Delete**.
6. Verify that the function list has been cleared successfully.

![Lambda Functions Cleared](/images/5-Workshop/5.18-CleanUp/Picture2.png)

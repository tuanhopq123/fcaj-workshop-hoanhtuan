   * Tìm kiếm và mở **Amazon API Gateway**.
   * Chọn **APIs**.

2. **Xóa API**
   * Trong danh sách, đánh dấu `energy-waste-http-api`.
   * Chọn **Actions** -> **Delete**.
   * Xác nhận xóa và chờ API biến mất khỏi danh sách.

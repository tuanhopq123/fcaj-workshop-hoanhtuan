1. **Open API Gateway**
   * Search for and open **Amazon API Gateway**.
   * Select **APIs**.

2. **Delete API**
   * Select `energy-waste-http-api`.
   * Click **Actions** -> **Delete**.
   * Confirm deletion and wait for it to disappear from the list.

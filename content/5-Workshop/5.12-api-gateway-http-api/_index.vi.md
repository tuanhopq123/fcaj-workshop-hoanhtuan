#### Tổng quan

Trong phần này, chúng ta sẽ tạo **HTTP API** bằng Amazon API Gateway. API này sẽ đóng vai trò là cổng tiếp nhận yêu cầu cho backend, xử lý các request từ dashboard và được bảo vệ bởi JWT authorizer.

1. **Quay lại AWS Console**
   * Vào `SNS → Topics → energy-waste-alert-topic → Subscriptions`.
   * Bấm refresh nếu cần.

2. **Kiểm tra trạng thái**
   * Bạn cần thấy trạng thái:

```
Status: Confirmed
```
